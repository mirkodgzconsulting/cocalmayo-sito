# Cocalmayo Clone

This is a modern clone of the Cocalmayo Coffee website, built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

## Stack
-   **Astro 5**: Static Site Generator
-   **Tailwind CSS v4**: Styling
-   **React**: UI Components
-   **Lucide React**: Icons

## Project Structure
-   `src/pages`: Website routes.
-   `src/components`: Reusable UI parts.
-   `src/layouts`: Main templates.
